# Project Statement: Student Attendance Manager

##  Problem Statement
The current manual method of tracking daily student attendance is inefficient, prone to errors, and time-consuming. This project aims to solve this by providing a simple, digital, and reliable system for recording and summarizing student presence.

##  Scope of the Project
The project is scoped as a **Command-Line Interface (CLI)** application focused on core attendance management.
* Student enrollment and management.
* Marking and tracking the cumulative count of days present.
* Data persistence via local JSON file storage.
* Displaying a summary report of attendance counts.

### Exclusions

##  Target Users
The primary users are **Instructors, Teachers, and Teaching Assistants** who require a quick and easy way to record student attendance for a class.

##  High-Level Features
1.  **Enrollment:** Add new students to the tracker.
2.  **Tracking:** Record daily presence status.
3.  **Persistence:** Save and load attendance data automatically.
4.  **Reporting:** View a total attendance count summary.
